静的コンテンツ（ハイドレーション不要）

このセクションはビルド時に生成された素のHTMLのみで構成されており、クライアント側のJavaScriptは一切読み込まれません。見出し・本文・画像といった静的な情報を表示するのに最適です。

Astroの設計思想

Astroはデフォルトでゼロランタイムを目指し、必要な箇所だけを選択的にハイドレートします。これにより配信されるJavaScriptの量を最小限に抑えられます。

導入事例の紹介

あるメディアサイトではAstroへの移行によって、初期表示に必要なJavaScriptを大幅に削減し、Core Web Vitalsのスコアを改善しました。ページの大部分は静的なままで、必要な部分だけが動的になります。

よくある質問（静的FAQ）

アイランドとは何ですか？
ページ内で個別にハイドレートされるインタラクティブなUIコンポーネントの単位です。
複数のフレームワークを同一ページで使えますか？
はい、React・Vue・Svelteなどのコンポーネントを混在させることができます。
SEOへの影響はありますか？
静的HTMLが中心のため、一般的にSEOに有利とされています。
既存プロジェクトへの導入は容易ですか？
段階的な移行がしやすい設計になっています。
