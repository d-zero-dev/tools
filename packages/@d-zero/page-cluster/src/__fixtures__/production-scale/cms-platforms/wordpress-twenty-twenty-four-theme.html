会社概要

サンプル合同会社は、2012年の設立以来、地域に根ざした造園・緑地管理事業を展開してまいりました。

会社概要

沿革

  • 2012年4月 — サンプル県サンプル市にて創業
  • 2015年7月 — 緑地管理事業部を新設
  • 2018年9月 — 神奈川支店を開設
  • 2021年3月 — 造園設計コンペにて優秀賞受賞
  • 2023年11月 — 本社ビルを現住所に移転

事業内容

造園設計・施工

個人邸から公共施設まで、幅広い規模の造園設計・施工を承っております。

緑地管理・剪定

年間契約による定期的な緑地管理、樹木剪定サービスを提供しています。

アクセス

サンプル線「サンプル前」駅より徒歩5分。サンプル線「サンプル」駅より徒歩8分。
