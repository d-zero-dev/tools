GET

/v1/tasks/{task_id}

指定したIDのタスク詳細を取得します。

MARKER_API_REFERENCE_DOCS_PAGE

パスパラメータ

名前型必須説明
task_idstring必須タスクの一意なID

クエリパラメータ

名前型必須説明
expandstring任意関連リソースを展開して取得
fieldsstring任意返却するフィールドを絞り込む

リクエスト例

curl -X GET https://api.cloudflow.example/v1/tasks/task_123 \
  -H "Authorization: Bearer YOUR_API_KEY"

レスポンス例

{
  "id": "task_123",
  "title": "デザインレビュー",
  "status": "in_progress",
  "assignee": "user_456",
  "due_date": "2026-07-10"
}

レスポンスフィールド

フィールド型説明
idstringタスクID
titlestringタスクタイトル
statusstringタスクのステータス
assigneestring担当者のユーザーID
due_datestring期限日（ISO 8601形式）

ステータスコード

コード説明
200取得成功
401認証エラー（APIキーが無効）
403アクセス権限がない
404指定したタスクが存在しない
429レート制限超過
500サーバー内部エラー

エラーレスポンス例

{
  "error": {
    "code": "not_found",
    "message": "指定されたタスクが見つかりません。"
  }
}
