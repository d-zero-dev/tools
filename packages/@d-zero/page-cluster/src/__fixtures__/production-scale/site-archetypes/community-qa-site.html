TypeScriptで条件付き型からユニオン型を抽出するには？

質問日: 2026-05-12 閲覧数: 3,412 編集日: 2026-05-14
  • typescript
  • generics
  • conditional-types
  • type-inference
42

条件付き型のテンプレートリテラル型から infer を使ってユニオン型を抽出したいのですが、うまくいきません。

type ExtractParam<T> = T extends `/${infer P}` ? P : never;

期待している挙動は以下の通りです。

  • 入力: /users/:id
  • 出力: :id のユニオン型
  • ネストしたパスにも対応したい

どなたか良い方法をご存知でしたら教えてください。

共有 編集 通報

  • 再帰的な条件付き型を使うと解決できそうです。

    – taro_dev

  • TSのバージョンは何を使っていますか？

    – hanako_ts

ベストアンサー

再帰的な条件付き型で分割していくのがシンプルです。

type Split<S> = S extends `${infer Head}/${infer Rest}` ? Head | Split<Rest> : S;
